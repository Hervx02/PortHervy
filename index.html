Présentation Du BTS SIO


Qu'est ce que le Bts Sio


Le BTS Services informatique aux Organisations est un diplôme reconnu par l'état de niveau Bac+2. Il remplace depuis septembre 2011 l'ancien BTS informatique de Gestion (BTS IG). En effet, cette nouvelle formule répond aux attentes de la profession en matière de qualification. Le programme a évolué en fonction de la place et du rôle des TIC (Technologies d'Information Communication) et propose deux spécialités bien distinctes : le BTS SIO SISR et le BTS SIO SLAM.Il associe aux enseignements théoriques et professionnels (en contrôle continu et final) deux stages en entreprise (10 à 11 semaines réparties sur les deux années détudes).En plus des matières en lien avec l'ingénierie informatique, la formation propose également des enseignements de culture générale, d'anglais (avec la possibilité de continuer une LV2), de mathématiques appliquées à l'informatique, ainsi que sur les outils d'analyse économique, managériale et juridique des services informatiques.De plus, au cours de cette formation, il est possible de bénéficier d'un enseignement approfondi en mathématiques et de travailler en autonomie en laboratoire informatique.

Le BTS SIO option SISR


L'option Solutions d'infrastructure, systèmes et réseaux forme des professionnels des réseaux et équipements informatiques (installation, maintenance, sécurité). En sortant d'un BTS SIO SISR, vous serez capables de gérer et d'administrer le réseau d'une société et d'assurer sa sécurité et sa maintenance.Les techniciens supérieurs en informatique option SISR, peuvent accéder aux métiers de : Administrateur systèmes et réseaux,Informaticien support et déploiement,Pilote d'exploitation,Support systèmes et réseaux,Technicien d'infrastructure,Technicien de production,Technicien micro et réseaux.

Le BTS SIO option SLAM


L'option Solutions logicielles et applications métiers forme des spécialistes des logiciels (rédaction d'un cahier des charges, formulation des besoins et spécifications, développement, intégration au sein de la société).Les techniciens supérieurs en informatique option slam, sont préparés aux métiers de Développeur d'applications informatiques Développeur informatique Analyste d'applications ou d'études Analyste programmeur Chargé d'études informatiques Informaticien d'études Programmeur analyste Programmeur d'applications Responsable des services applicatifs Technicien d'études informatique

Mes Competences

CSS

CSS est l'acronyme de Cascading Style Sheets, qui signifie en français feuilles de style en cascade. C'est un langage informatique utilisé pour décrire la présentation visuelle d'une page web, en appliquant des styles aux éléments HTML. Les styles peuvent inclure des propriétés telles que la couleur, la taille de police, la mise en page, les bordures, les arrière-plans, les effets de transition, etc. CSS permet de séparer la présentation visuelle de la structure HTML d'une page web, ce qui facilite la maintenance et la mise à jour du site.

PHP

PHP est un acronyme récursif qui signifie en anglais "PHP: Hypertext Preprocessor". C'est un langage de programmation côté serveur utilisé pour développer des applications web dynamiques. Il est principalement utilisé pour générer des pages web dynamiquement à partir de bases de données et pour interagir avec des formulaires web. PHP est souvent utilisé en conjonction avec HTML et CSS pour créer des sites web dynamiques et interactifs. Il est open source et gratuit, et est compatible avec une grande variété de systèmes d'exploitation et de serveurs web.

HTML

HTML est l'acronyme de HyperText Markup Language, qui signifie en français langage de balisage hypertexte. C'est un langage informatique utilisé pour créer et structurer le contenu des pages web, en utilisant des balises pour indiquer la signification et la mise en forme du texte, des images et d'autres éléments. En d'autres termes, HTML permet de créer des pages web en définissant leur contenu et leur structure..

JS

JS est l'abréviation de JavaScript, qui est un langage de programmation côté client utilisé pour créer des fonctionnalités interactives sur les pages web. JavaScript est souvent utilisé pour ajouter des effets visuels, des animations, des pop-ups, des menus déroulants et des formulaires dynamiques à une page web. Il est également utilisé pour la validation des formulaires, la manipulation de contenu HTML et pour la création de jeux en ligne. JavaScript est un langage interprété, ce qui signifie qu'il est exécuté directement dans le navigateur web de l'utilisateur. Cela le rend très rapide et réactif, et permet aux développeurs de créer des expériences utilisateur très dynamiques et interactives.

C#

C# (prononcé "C Sharp") est un langage de programmation orienté objet développé par Microsoft dans les années 2000. Il est principalement utilisé pour créer des applications pour la plate-forme Microsoft .NET, notamment des applications de bureau, des applications Web, des jeux et des applications mobiles. C# est souvent utilisé en conjonction avec d'autres technologies Microsoft, telles que le framework .NET, ASP.NET, Xamarin et Unity pour le développement de jeux. C# a une syntaxe similaire à celle de C++ et de Java, ce qui le rend facile à apprendre pour les programmeurs ayant une expérience préalable dans ces langages.

Symphony

Symfony est un framework de développement d'applications web en PHP. Il est open-source et a été créé en 2005 par Sensio Labs. Symfony est basé sur le modèle de conception MVC (Modèle-Vue-Contrôleur) et offre une grande flexibilité pour le développement d'applications web complexes. Il propose un grand nombre de composants réutilisables et de bonnes pratiques pour le développement de projets PHP.

MYSQL

MySQL est un système de gestion de base de données relationnelle (SGBDR) open-source qui utilise le langage SQL (Structured Query Language) pour gérer et stocker des données. Il a été créé en 1995 et est l'un des SGBDR les plus populaires et les plus utilisés au monde. MySQL est utilisé pour stocker des données dans des applications web et des sites internet, et peut gérer de grands volumes de données avec une grande efficacité. Il est souvent utilisé en conjonction avec des langages de programmation tels que PHP, Python, Ruby, Java et C#.

SQL

SQL (Structured Query Language) est un langage de programmation utilisé pour communiquer avec des systèmes de gestion de bases de données relationnelles (SGBDR) tels que MySQL, Oracle, Microsoft SQL Server et PostgreSQL. Il est utilisé pour créer, modifier, supprimer et interroger des bases de données relationnelles. SQL permet aux utilisateurs de définir des schémas de base de données, de créer des tables, de définir des relations entre les tables, d'insérer des données, de mettre à jour des données, de supprimer des données, de trier des données, de filtrer des données et d'effectuer des jointures entre plusieurs tables. Les utilisateurs peuvent également utiliser des fonctions SQL pour effectuer des calculs sur les données stockées dans la base de données. SQL est un langage très important pour les développeurs et les administrateurs de bases de données, car il permet de gérer efficacement les données dans les applications et les sites web.

Boostrap

Bootstrap est un framework de développement web open-source créé par Twitter en 2011. Il est utilisé pour créer des sites web responsives et adaptatifs, ce qui signifie que les sites web développés avec Bootstrap s'adaptent automatiquement à différents types d'appareils, tels que les ordinateurs de bureau, les tablettes et les smartphones. Bootstrap fournit un ensemble de styles CSS prédéfinis, des composants JavaScript et HTML pour la création de boutons, de menus déroulants, de formulaires, de modèles de grille de mise en page et bien plus encore. Il permet aux développeurs de créer rapidement des sites web avec un design cohérent et professionnel. Bootstrap est très populaire et est utilisé par de nombreuses entreprises, notamment Airbnb, Spotify, LinkedIn et Amazon..
